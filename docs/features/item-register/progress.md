## 블로커 / 이슈 / 특이사항

- **분류 모델 v1 미선정.** T008에서 MediaPipe·CLIP·자체 학습 중 PoC 비교 필요. 결정 전까지 ClassifierService는 null stub으로 동작(category=null로 응답).
- **백그라운드 큐 인프라 미선정.** T017이 in-memory로 시작할지 BullMQ/Redis로 갈지 결정 필요.
- **DB 마이그레이션 미적용.** Prisma 모델만 추가됨. T015 ItemBatch는 미정의. `prisma migrate dev` 1회 실행 필요(`closet` 슬러그가 이미 Item 모델 보유 중).
- **P1만 구현됨.** T001~T007 완료. P2~P6은 모두 대기 상태. 다음 세션에서 우선순위 정해 이어가야 함.
