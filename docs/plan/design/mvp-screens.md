# My Closet MVP — 기능·화면 목록

## 1. 로그인·온보딩

**기능**
- 휴대폰 번호로 가입·로그인
- SMS 인증번호 확인
- PIN 6자리 설정·로그인
- 생체인식 등록·로그인
- PIN 재설정
- 휴대폰 번호 변경

**화면**
- 온보딩 동의 화면
- 휴대폰 번호 입력 화면
- SMS 인증번호 입력 화면
- PIN 설정 화면
- PIN 로그인 화면
- 생체인식 등록 다이얼로그
- PIN 재설정 화면
- 휴대폰 번호 변경 화면

---

## 2. 홈

**기능**
- 오늘의 코디 확인
- 주요 기능 빠른 진입(옷 등록·코디 만들기·캘린더)

**화면**
- 홈/대시보드 화면
- 빈 옷장 상태 화면 (0벌일 때)

---

## 3. 옷 등록

**기능**
- 카메라로 옷 한 벌 촬영해 등록
- 갤러리에서 한 장 선택해 등록
- 갤러리에서 최대 20장 멀티 선택해 일괄 등록
- AI 자동 분류 (카테고리·색상)
- AI 결과 1탭 수정

**화면**
- 카메라/갤러리 선택 화면
- 촬영 화면 (가이드 프레임)
- 사진 미리보기·재촬영 화면
- AI 분류 결과·수정 화면
- 멀티 선택 화면 (갤러리)
- 배치 업로드 진행 화면

---

## 4. 옷장 둘러보기

**기능**
- 등록한 옷 그리드로 보기
- 카테고리·색상 필터
- 검색
- 무한 스크롤
- 사진 뷰 ↔ 이모지 뷰 전환
- 옷 상세 보기·수정·삭제

**화면**
- 옷장 그리드 화면 (사진 뷰)
- 옷장 그리드 화면 (이모지 뷰)
- 옷 상세 화면

---

## 5. 코디 만들기

**기능**
- 마네킹 4슬롯(상의·하의·아우터·신발)에 옷 드래그&드롭
- 코디 저장
- 셔플(랜덤 코디 제안)
- 저장된 코디 목록 보기·삭제

**화면**
- 마네킹 코디 보드 화면
- 코디 셔플 결과 화면
- 저장된 코디 목록 화면
- 코디 상세 화면

---

## 6. 캘린더

**기능**
- 월 단위 캘린더에서 날짜별 코디 확인
- 특정 날짜에 "오늘 입기" 기록
- 과거 회상 뷰

**화면**
- 월 캘린더 화면
- 날짜 상세 화면
- "오늘 입기" 기록 화면

---

## 7. 공유

**기능**
- OOTD 이미지 합성 (마네킹·옷·워터마크·딥링크 QR)
- 인스타·카톡 등 시스템 공유 시트로 내보내기

**화면**
- OOTD 공유 미리보기 화면
- 시스템 공유 시트 (네이티브)

---

## 8. 동기화

**기능**
- 클라우드 자동 업로드
- 새 기기에서 로그인 시 옷장 복원
- 오프라인 큐 처리

**화면**
- 동기화 상태 화면

---

## 9. 설정

**기능**
- 계정 관리
- 데이터 내보내기 (내 옷장 파일로 받기)
- 로그아웃

**화면**
- 설정 화면
- 데이터 내보내기 화면

---

## 10. 공통 디자인 자산

- 색상 팔레트 (라이트/다크)
- 타이포·간격·라운드 토큰
- 카테고리 아이콘 (상의·하의·아우터·신발 등)
- 액션 아이콘 (카메라·필터·셔플·공유 등)
- 카테고리+색상 → 이모지 매핑표
- 빈 상태 일러스트 (옷장 0벌, 코디 0개, 캘린더 빈 날)
- 로딩·에러 상태 (AI 분류 중·업로드 중·네트워크 실패·권한 거부)
- 토스트·다이얼로그 스타일
- 앱 아이콘·스플래시 (iOS/Android)
