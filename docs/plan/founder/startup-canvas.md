## Part 1. Product Strategy

### 1. Vision

옷장을 열기 전, 앱을 먼저 엽니다. 사진 한 장으로 시작해서 매일의 옷 고민을 끝내는 가장 가벼운 디지털 옷장이 되는 것이 목표입니다.

### 2. Market Segments (JTBD 중심)

| 우선순위 | Segment | JTBD |
|---------|---------|------|
| 🔴 Beachhead | S1. SNS 패션러 (20대 여성) | 거울 앞에서 시간을 보낼 때, 새로운 코디를 빠르게 발견하고 잘 나온 룩을 SNS에 기록·공유하고 싶다. |
| 🟡 인접 확장 | S2. 결정 피로 직장인 (30대) | 출근 준비 중일 때, 오늘 날씨·일정에 맞는 옷을 빠르게 결정하고 더 이상 고민하고 싶지 않다. |
| 🟡 유료화 타겟 | S4. 절약·가족관리자 (30~40대) | 쇼핑 직전, 비슷한 옷을 이미 가지고 있는지 확인해 충동구매를 막고 싶다. |
| 🟢 후순위 | S3. 미니멀·정리광 | 옷장 전체 상태를 한눈에 파악하고, 안 입는 옷을 정리·기부·중고판매하고 싶다. |

**First segment.** S1 (SNS 패션러). 가장 높은 사용 빈도, 자발적 SNS 확산, 등록 의지가 기대됩니다.

### 3. Relative Costs (저원가 vs 차별화 가치)

| 우리가 낮게 가는 비용 | 우리가 높게 가는 가치 |
|---------------------|---------------------|
| 마케팅 — 인플루언서 시드 + 바이럴 콘텐츠 중심, 유료 광고 최소화 | UX 디자인 — "가벼움"과 "발견의 즐거움"을 정체성으로 |
| 인프라 — 온디바이스 추론으로 서버 비용 절감 | AI 분류 정확도 — 80% Top-1 이상이 모든 가치의 전제 |
| 영업·CS — 셀프 서브, 커뮤니티 기반 지원 | 한국 의류 도메인 적합도 — 한국어·날씨·핏 1st |

> 포지션. **저원가 + 차별화 동시 추구 (Hybrid)**. 저원가 인프라 위에 차별화된 UX와 한국 도메인 깊이를 얹습니다.

### 4. Value Proposition

**For (Who).** 거울 앞에서 매일 코디를 고민하는 20대 SNS 사용자.

- **What Before.** 옷은 가득한데 입을 게 없습니다. 같은 조합만 반복하고, 인스타에서 영감을 저장해도 내 옷에 적용이 안 됩니다.
- **How.** 옷을 사진으로 등록하면 AI가 자동 분류합니다. 이모지/사진 모드로 한눈에 보고, 드래그·셔플로 코디를 발견합니다. 매일 아침 날씨에 맞는 추천을 받고, 1탭으로 OOTD를 공유합니다.
- **What After.** 옷 고민 시간이 절반으로 줄고, 매주 새로운 조합을 발견합니다. 잘 입는 옷 데이터로 다음 쇼핑이 똑똑해집니다.
- **Alternatives.** 거울 셀카 + 인스타 OOTD 폴더 + 머릿속 기억. 우리는 "내 옷 데이터" 위에서 동작한다는 점이 다릅니다.

**한 줄 문장.**
> "옷 고민이 매일 5분씩인 20대를 위해, My Closet은 사진 한 장으로 시작하는 디지털 옷장입니다. Acloset과 달리, 무료에서 충분히 가볍게 시작할 수 있고 한국 SNS 흐름에 맞춰져 있습니다."

### 5. Trade-offs (의도적으로 하지 않는 것)

| 선택 | 포기 | 이유 |
|------|------|------|
| 한국어·한국 우선 | 글로벌 동시 출시 | 좁고 깊게 이깁니다. 한국에는 Acloset 외 적수가 없습니다. |
| 갤러리 일괄 import + 5벌 챌린지 | AI 분류 완벽주의 | 등록 마찰이 모든 가치의 관문입니다. |
| 무료 풍부 + 유료는 P4 가드 모듈 | 모두에게 100벌 같은 임의 벽 | Acloset 약점을 정면 공략합니다. |
| 코디 발견 + SNS 공유 | 캡슐 코칭, 인간 스타일리스트 | Cladwell 영역에 진입하지 않습니다. |
| 1인 옷장 → v1.5에서 가족 공유 | v1 가족·소셜 옷장 | 1인 가치 검증이 선결입니다. |
| 모바일 First (iOS+Android) | iOS Only | Z세대 Android 비중을 포기할 이유가 없습니다. |

### 6. Key Metrics

- **North Star.** WAU당 코디 생성 수 (Coordinated Outfits per WAU).
- **OMTM (지금 한 달).** 신규 가입자의 7일 내 등록 옷 평균 수 (목표 ≥20벌).
- **Input Metrics.**
  - D7 retention ≥30%.
  - WAU 중 코디 보드 진입률 ≥50%.
  - OOTD 공유 클릭률.
  - 중복구매 경고 → 클릭 전환율 (유료화 시그널).
- **Health.** 등록 평균 시간 ≤8초/장, AI 보정률 ≤20%, 크래시율, P95 응답시간.

### 7. Growth (어떻게 성장할 것인가)

- **모션.** Product-Led Growth + 콘텐츠·인플루언서 마케팅.
- **Acquisition.** 인스타·틱톡 릴스 ("옷장 1분 정리 챌린지"), 패션 인플루언서 50명 시드 베타, ASO ("옷장 정리", "오늘 뭐 입지").
- **Activation.** 갤러리 일괄 import + 등록 N벌 도달 시 첫 코디 자동 제안. (강제 5벌 챌린지는 v0.2에서 MVP 제외, v1.5 재기획.)
- **Retention.** "어제 뭐 입었지" 회상(P1-03 위젯) + 캘린더 기록 + 셔플 재실행. (날씨 추천 푸시는 v1.5에서 발송 타이밍 모델 정의 후 도입.)
- **Referral.** 1탭 OOTD 공유에 앱 링크 워터마크, 친구 옷장 공유 (v1.5).
- **Monetization.** v1.5에 P4 타겟 "쇼핑 가드" 프리미엄 (월 4,900원).

### 9. Can't / Won't (경쟁사가 못 하거나 안 하는 것)

- **Acloset이 못 한다.** 무료 100벌 벽을 풀면 자사 수익 모델이 흔들립니다. 우리는 처음부터 무료 풍부 모델로 설계 가능합니다.
- **Whering·Indyx가 안 한다.** 한국 의류·날씨·한국어 1st UX. 우선순위가 글로벌이라 한국 시장 깊이 진입할 동기가 약합니다.
- **Stylebook이 못 한다.** AI 자동 분류, 모바일 First, Android 지원. 15년 된 코드베이스와 1인 개발 체제 한계.
- **Cladwell이 안 한다.** 캡슐 코칭이 아닌 자유로운 코디 발견. 포지셔닝과 충돌합니다.

## Part 2. Business Model

### Cost Structure

**고정비.**
- 개발 인건비 (모바일 2 + 백엔드 1 + 디자이너 1, 초기 가정).
- 클라우드 인프라 기본료.
- 도구·라이선스 (Figma, Sentry, GA 등).

**일회성 / 초기 투자비 (Pre-MVP).**
- **한국 의류 라벨링 데이터셋 구축.** 8개 카테고리 × 1,500장 = 약 12,000장 라벨링. 외주 라벨러 비용 ~₩3,000,000 (장당 250원 기준). AI 분류 정확도 80% 달성의 핵심 자본.
- AI 모델 fine-tune GPU 비용 (1회성, ~₩500,000).
- 마네킹 일러스트 자체 제작 또는 라이선스 (~₩1,000,000).
- 법무 자문 (약관·개인정보·14세 차단) (~₩1,500,000).

**변동비 (사용자 수에 비례).**
- 이미지 스토리지 (사용자 1인당 옷 100~300장 × 평균 200KB).
- AI 추론 비용 (온디바이스 우선, 일부는 서버 추론).
- 푸시·SMS 비용.
- 결제 수수료 (Apple/Google 15~30%).

**지속 운영비 (출시 후).**
- AI 회귀 셋 라벨링 유지 (월 ~₩200,000, 사용자 보정 데이터 활용 시 점진적 감소).
- 인플루언서 시드 인센티브 (월 ~₩2,000,000~5,000,000).

**한 사용자당 추정 한계비용.** 월 100~300원 수준 (스토리지·추론·푸시 합산). 무료 사용자도 흑자에 큰 부담은 아닙니다.

### Revenue Streams

| 단계 | 매출원 | 가격 | 비고 |
|------|-------|------|------|
| MVP (0~6m) | 없음 (성장 우선) | — | 사용자·데이터 확보가 목표 |
| v1.5 (6~12m) | 프리미엄 구독 | ₩4,900/월 또는 ₩39,000/년 | P4 타겟 "쇼핑 가드" 패키지 |
| v2 (12~18m) | 가족 플랜 | ₩7,900/월 (3~4인) | 가족 옷장 공유 + 가드 |
| v2 (12m+) | 광고 (선택적) | CPM 기반 | 무료 사용자 한정, 패션 브랜드만 |
| v2.5 (18m+) | 제휴 매출 (커머스 연동) | 매출의 5~10% | 무신사·29CM 영수증 import 파트너십 |
| v3+ | 데이터·B2B 라이선스 | 협의 | 익명화된 한국 옷장 통계 |

### Strategy Coherence Check

- **Vision ↔ Trade-offs.** "가벼운 옷장" 비전과 "갤러리 일괄 import / 5벌 챌린지" 트레이드오프가 일치합니다.
- **Segments ↔ Capabilities.** S1 비치헤드에 필요한 SNS Export·코디 보드가 MVP에 모두 포함되어 있습니다.
- **Value Prop ↔ Revenue.** 무료에서 핵심 가치(코디 발견)를 충분히 제공하고, P4 타겟의 명확한 ROI 가치(중복구매 방지)에서만 결제를 요구해 일관됩니다.
- **Growth ↔ Cost.** PLG + 콘텐츠 중심으로 CAC를 낮게 유지하며, 한계비용도 낮아 PLG 흑자 구조가 가능합니다.
- **한국 1st ↔ AI 자본.** 한국 의류 라벨링 데이터 ~₩3M, fine-tune·일러스트·법무 포함 Pre-MVP 일회성 비용 ~₩6M을 Cost Structure에 명시화 완료. 자본 조달 또는 단계적 라벨링(MVP는 5,000장으로 시작)으로 분할 가능.

### Riskiest Assumptions

1. **사용자가 옷을 사진으로 등록할 의지가 있다 (V1).** → 컨시어지 실험(E1)으로 검증.
2. **AI 분류 정확도 Top-1 80% 달성 가능 (F1).** → 오픈 모델 PoC(E4)로 검증. NO-GO 시 전체 전략 재검토.
3. **P4(절약형) 사용자가 월 4,900원에 결제할 의향이 있다.** → 페이크 도어 결제 버튼 + Wizard-of-Oz(E5)로 검증.
4. **인플루언서 시드 50명으로 자발적 SNS 확산이 일어난다.** → 페이크 도어 LP(E2) + 베타 시작 후 4주 관찰.
5. **온디바이스 추론으로 변동비를 충분히 낮출 수 있다.** → 기술 PoC, 디바이스별 성능 측정.
