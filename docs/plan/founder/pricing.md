# Pricing Strategy. My Closet

- **Date.** 2026-05-17
- **Stage.** Pre-MVP (구독 도입 v1.5 시점 검증 필요).
- **관련 문서.** [startup-canvas.md](startup-canvas.md), [north-star.md](north-star.md), [../researcher/competitor-analysis.md](../researcher/competitor-analysis.md)

## 1. 가격 전략 원칙

1. **MVP는 전부 무료.** Active Outfit·retention·바이럴 데이터 확보가 매출보다 100배 가치 있다.
2. **결제 트리거는 명확한 ROI 시점에서만.** P4(절약·가족관리자) 페르소나의 "중복구매 1회 = 월 구독 12개월 본전" 계산이 성립할 때만 결제를 요구한다.
3. **임의의 옷 수 벽 금지.** Acloset의 100벌 벽은 등록 임계(30벌) 직전에 결제 압박을 거는 자살 정책. 우리는 옷 등록·기본 코디·캘린더·공유를 평생 무료로 둔다.
4. **한국 시장에 맞는 가격대.** 글로벌 평균 $6~10/월보다 낮은 ₩4,900/월 (≈$3.5) 진입가로 가격 저항을 줄인다.

## 3. 가격 모델 비교

| 모델 | 장점 | 단점 | 적합도 |
|------|------|------|--------|
| **Freemium + 프리미엄 구독** (추천) | 진입 마찰 0, 데이터 축적 후 자연 전환 | 전환율 1~5% 가정 필요 | ★★★★★ |
| 일회성 결제 (Stylebook 형) | 구독 거부층 흡수, 단순 | LTV 낮음, 추가 매출 어려움 | ★★ |
| 옷 수 기반 단계형 (Acloset 형) | 명확한 트리거 | 등록 임계 직전 결제 압박 = 자살 | ★ |
| 광고 모델 단독 | 사용자 비용 0 | 패션 광고 매칭 어려움, 사용 경험 저하 | ★★ |
| 거래 수수료 (커머스 제휴) | 사용자 비용 0 | 파트너십 의존, 출시 후 18개월+ | ★★★ (보조 매출) |

**선택. Freemium + 프리미엄 구독 (1차 매출) + 거래 수수료 (보조 매출, v2.5+)**

## 4. 가격 구조 (v1.5 도입 시점)

### Free 티어 (영구 무료)
- 옷 등록 무제한.
- 기본 코디 도구 (마네킹·셔플·저장).
- 캘린더 기록.
- 날씨 기반 일일 추천 푸시.
- OOTD 1탭 SNS 공유.
- 옷별 기본 통계 (착용 횟수, 마지막 착용일).
- 데이터 동기화·복구.

### Premium 티어 — "쇼핑 가드" (₩4,900/월 또는 ₩39,000/년, 33% 할인)
- **중복구매 경고** — 위시리스트·쇼핑몰 URL 입력 시 보유 옷과 이미지 유사도 비교.
- **가족·친구 옷장 공유** (3인까지).
- **고급 통계** — 컬러 분포, 활용률 점수, 안 입은 옷 자동 정리 추천, 월간 스타일 리포트.
- **광고 제거**.
- **여행 패킹 리스트 자동 생성** (일정·날씨 입력 시).

### Family Plan (₩7,900/월 또는 ₩63,000/년)
- Premium 전 기능 + 가족 옷장 5인까지 + 자녀 옷 사이즈 추적.

## 5. 가격 민감도 (가설)

- **₩2,900/월.** 너무 저렴해서 가치 인식 저하 우려, 광고 모델과 차별화 약함.
- **₩4,900/월.** 💚 진입가로 권장. 카카오톡 이모티콘 ~2개 가격.
- **₩7,900/월.** 가족 플랜에 적합. 단독 개인 플랜으로는 P4가 망설일 수 있는 임계.
- **₩9,900/월 이상.** Indyx 글로벌 가격대. P4 ROI는 여전히 성립하나 한국에서는 거부감 큼.

검증 필요. **E5(WoZ 날씨 추천)** 마지막 주에 "월 4,900원이면 유료 사용 의향?" 직접 인터뷰로 1차 확인. 정식 출시 후 가격 A/B (₩4,900 vs ₩6,900) 권장.

## 7. 매출·전환 가설

- **무료 → 유료 전환율.** 1년차 1.5%, 2년차 3% (산업 평균 2~5%).
- **무료 사용자당 월 한계비용.** ₩100~300 (스토리지·추론).
- **유료 사용자당 월 매출.** ₩4,900 (개인) / ₩7,900 (가족, v2부터).
- **결제 수수료.** Apple/Google IAP 15~30%, 토스 결제 1.7~3.3%.

## 8. 가격 리스크

| Risk | Likelihood | Impact | Mitigation |
|------|:---------:|:------:|------------|
| Acloset이 무료 정책을 대폭 확장 | 중 | 큼 | 무료에서도 핵심 가치(코디 발견)를 끝까지 무제한 제공해 가격이 아닌 UX·도메인 적합도로 차별화. |
| P4 페르소나의 결제 의향이 가설보다 낮음 | 중-상 | 치명 | E-P1·E-P2 우선 실행, 미달 시 광고·커머스 제휴 모델로 무게중심 이동. |
| Apple/Google 수수료가 가격의 30%를 가져감 | 상 | 중 | 연 구독 비율 ≥30% 유도(Apple 15%로 인하 적용), 자체 결제 옵션 검토. |
| 한국 사용자가 구독 자체에 저항 | 중 | 중 | 일회성 결제 옵션(예. ₩29,000 1년치 패키지) 병행 검토. |
| 가격 인하 경쟁이 시작됨 | 저 | 중 | 가격 경쟁 진입 거부, 가족 플랜·고급 통계로 가치 차별화. |

## 9. 결정 요약

- v1.5에 **₩4,900/월 (또는 ₩39,000/년) 단일 프리미엄 도입**.
- 결제 트리거는 **"쇼핑 가드"** (P4 페르소나의 ROI 명확 기능).
- E-P1·E-P2를 먼저 통과해야 가격 도입 GO.
- v2에 가족 플랜(₩7,900/월) 추가, v2.5+에 커머스 제휴 매출 도입.
