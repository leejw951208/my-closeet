## 1. Beachhead Segment

### Who
**한국 거주 20~26세 여성 중 인스타·핀터레스트에 OOTD 폴더를 가진 패션 액티브 사용자.** 옷장 40~80벌 보유, 매일 5~15분을 옷 결정에 사용하며 SNS에 본인 룩을 주 1회 이상 공유합니다.

### Why them first (4-기준 평가)
| 기준 | 평가 | 메모 |
|------|------|------|
| Burning pain | 🔴 상 | "옷은 많은데 입을 게 없다"의 매일 반복, 1020 여성층에서 가장 강함. |
| Willingness to pay | 🟡 중 | 직접 결제 의향은 낮으나 광고·인플루언서 마케팅 흡수율 높음. |
| Winnable market share | 🟢 상 | Acloset이 한국 단독이나 가격 벽·UX 식상함이 약점. 정면 우회 가능. |
| Referral potential | 🔴 상 | OOTD·옷장 자랑이 콘텐츠 자체. 자발 확산 잠재력 최강. |

### Size (개략 추정)
- **TAM (한국 1020·2030 여성 스마트폰 사용자).** 약 800만~1,000만 명.
- **SAM (패션 관심 + SNS 액티브).** 약 250만~400만 명.
- **SOM (12개월 내 도달 가능).** 약 10만~30만 MAU 목표 (Acloset 한국 추정치의 5~10%).

## 3. Positioning & Messaging

### Positioning Statement

**옷 고민이 매일 5분씩인 20대 여성을 위해, My Closet은 사진 한 장으로 시작하는 가벼운 디지털 옷장입니다. Acloset과 달리, 무료에서 충분히 가볍게 시작할 수 있고 한국 SNS 흐름에 맞춰져 있습니다.**

### Key Messages (스테이크홀더별)

| Audience | Message | Proof Point |
|----------|---------|------------|
| 잠재 사용자 (Primary) | "옷장에 옷은 가득한데 입을 게 없을 때, 우리가 새 조합을 찾아드려요." | 첫 5벌 등록만으로 코디 자동 생성 데모 영상 |
| 잠재 사용자 (Primary) | "사진 한 장이면 끝. 갤러리 20장 한꺼번에 가져오기." | 등록 8초 / 일괄 import 1분 시연 |
| 잠재 사용자 (Primary) | "오늘 날씨에 맞는 코디, 매일 아침 9시에 도착해요." | 푸시 알림 스크린샷 |
| 인플루언서 | "본인 옷장으로 OOTD 콘텐츠를 더 쉽게 만들어보세요. 워터마크는 본인 브랜딩으로 커스텀 가능합니다 (인플루언서 한정)." | 시드 베타 50명 한정 인센티브 |
| 미디어·블로거 | "한국 사용자에 맞춰 다시 만든 옷장 앱. 무료에서 다 됩니다." | Acloset 비교표, 한국어·날씨·핏 차별점 |
| 패션 커머스 파트너 (v1.5+) | "당신의 구매 영수증을 옷장으로 자동 연결. 사용자 retention과 재구매 데이터를 함께." | API 사양서, 파일럿 데이터 |

## 8. Expansion Plan (Post-Beachhead)

### Stage 1. 비치헤드 인접 확장 (출시 후 3~6개월)
- **타겟.** P2 결정 피로 직장인 (27~35세 여성·남성).
- **변화.** "어제 뭐 입었지" 위젯(P1-03)과 푸시 시간 설정(P1-04)을 전면 배치. 광고 카피를 "출근 5분 컷 코디"로 추가.
- **채널.** 카카오 채널, 직장인 커뮤니티(블라인드·잡플래닛 매거진).

### Stage 2. 유료화 진입 (출시 후 6~9개월)
- **타겟.** P4 절약·가족관리자 (30~40대 부모층).
- **상품.** v1.5 "쇼핑 가드" 구독 (월 4,900원). 중복구매 경고 + 가족 옷장 공유.
- **채널.** 맘카페·육아 커뮤니티·네이버 카페 시드. 무신사·29CM 영수증 import 파트너십 (필요 시).

### Stage 3. 데이터·B2B (출시 후 9~18개월)
- **타겟.** 패션 커머스·스타일링 서비스·중고 거래 플랫폼.
- **상품.** 익명화된 한국 옷장 트렌드 데이터 라이선스, 영수증 import 파트너 API.

### Stage 4. 글로벌 (출시 후 18개월+)
- 일본·동남아 우선. 한국 한류·K-패션 흐름을 활용. 다국어·통화 대응 후 진입.
